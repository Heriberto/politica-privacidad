Política de Privacidad

Esta aplicación utiliza la información que el usuario comparte a través de Messenger únicamente para interactuar con el bot y brindar respuestas automatizadas.

No almacenamos de manera permanente datos personales sensibles, ni compartimos información con terceros.

El uso de esta aplicación está destinado únicamente a mejorar la comunicación con el usuario y brindar un mejor servicio.

Al interactuar con nuestro bot, el usuario acepta que sus mensajes sean procesados automáticamente para generar una respuesta.
